# ordinal-suffix

A simple little module to add ordinal suffixes to numbers

### installation
```bash
npm install --save ordinal-suffix
```

### example
```js
var ordinal = require('ordinal-suffix')

// Output: 1st
ordinal(1)

// Output: 23rd
ordinal(23)

// Output: 70th
ordinal(70)

// Output: 92nd
ordinal(92)
```
